CoderBot - A programmable robot with educational purpose

Easy to use

No need to be an expert in Information Technology or Electronics to be able to use it.

More than a toy

Learn logic, coding skills and much more, having fun.

Fit for School

CoderBot is suited for use in the classroom with groups of kids.

Free and Open source

CoderBot is an open source and open hardware project.

Maker friendly

Download the schematics and build your very own CoderBot in a FabLab or Makerspace.

Extensible

Modify, hack, extend CoderBot, build your very personal bot.

Features

Control it by Computer, Tablet, Smartphone

CoderBot scan be controlled and programmed by any device running an html5 compatible browser and wifi.

Coding? Kid's play

The programming interface is configurable for different levels of complexity, suitable for kids from 6 and up.

Computer Vision: video streaming and object detection

The videocamera can stream live images and can be used to detect the environment, objects, obstacles, paths.

Sound and speech synthesis

CoderBot che communicate using the speaker: predefined sounds and synthesised speech
